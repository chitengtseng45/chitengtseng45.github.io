# 兒科肌肉骨骼超音波教學網

由 **曾吉騰醫師（Chi-Teng Tseng, MD-MPH）** 主持的兒科肌肉骨骼超音波工作坊延伸學習平台。
基隆長庚紀念醫院 兒童過敏氣喘風濕科 (AIR)。

**線上網址**：<https://chitengtseng45.github.io/>

---

## 網站內容

| 區塊 | 檔案 | 說明 |
| --- | --- | --- |
| 首頁 | `index.html` | Hero、四大教學模組、延伸資源 |

## 技術說明

- 純靜態網站（HTML / CSS / JavaScript），無需後端或資料庫。
- 以 GitHub Pages 免費託管。推送到 `main` 分支後 1–2 分鐘自動部署。
- 手機體驗由 `js/site-enhance.js` 強化：漢堡選單、目錄收合、表格橫向滑動、電腦／手機切換。
